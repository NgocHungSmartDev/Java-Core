# Lambda expressions: Java 8 hỗ trợ các biểu thức lambda, cung cấp cách viết mã ngắn gọn hơn và dễ đọc hơn.

# Functional interfaces: Java 8 định nghĩa một số giao diện chức năng để hỗ trợ việc sử dụng biểu thức lambda.

# Stream API: Stream API cho phép bạn thao tác với các bộ dữ liệu trong Java một cách linh hoạt và dễ dàng hơn.

# Date and Time API: Java 8 cung cấp một API mới để xử lý ngày tháng, giúp cho việc xử lý ngày tháng trở nên dễ dàng hơn.

# Default methods: Java 8 cho phép giao diện xác định một phương thức mặc định để hỗ trợ sự phát triển chéo.

# Method references: Java 8 cung cấp cú pháp viết ngắn gọn để tham chiếu đến một phương thức đã được xác định trước đó.

# Optional: Optional cho phép bạn xử lý các giá trị có thể null một cách an toàn hơn.

# Ngoài ra, để hiểu Java 8 tốt hơn, bạn cũng nên tìm hiểu về các tính năng khác như parallel streams, collectors, và các cải tiến khác của ngôn ngữ Java.
