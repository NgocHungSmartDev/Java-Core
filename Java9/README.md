# Java Platform Module System (JPMS): Java 9 cung cấp một cơ chế mới để quản lý các module trong Java. Điều này giúp người phát triển tạo ra các ứng dụng Java có kích thước nhỏ hơn, đồng thời tăng tính bảo mật và khả năng tái sử dụng mã nguồn.

# JShell: JShell là một công cụ REPL (Read-Eval-Print-Loop) cho phép người dùng tương tác với Java thông qua các lệnh dòng lệnh, giúp cho việc thử nghiệm và phát triển trở nên nhanh hơn.

# G1 Garbage Collector: Java 9 cung cấp một bộ thu gom rác mới gọi là G1, giúp cải thiện hiệu suất và tăng khả năng đáp ứng của ứng dụng.

# Reactive Streams: Java 9 hỗ trợ Reactive Streams, một công nghệ mới giúp quản lý các dòng dữ liệu một cách hiệu quả, đồng thời giảm thiểu tình trạng tràn bộ đệm.

# Javadoc Search: Java 9 cung cấp một tính năng tìm kiếm Javadoc, giúp người dùng nhanh chóng tìm kiếm các thông tin liên quan đến các lớp và phương thức trong mã nguồn Java.

# HTTP/2 Client: Java 9 hỗ trợ HTTP/2, giúp tăng tốc độ truyền tải dữ liệu giữa các ứng dụng web.

# Ngoài ra, Java 9 còn cung cấp nhiều tính năng khác như Stack-Walking API, Multi-Release JAR Files, Enhancements to Streams API, Process API Updates, và nhiều cải tiến khác giúp cho người phát triển có thể tạo ra các ứng dụng Java tốt hơn.
